# Hangman
Jeu de devine mot

Ceci est un projet basique du jeu hangman qui consiste à deviner le mot exacte.

J'ai choisi des mots au hasard puis je les ai mis dans mon code. Ensuite j'ai utilisé la fonction random pour les choisir aléatoirement puis le jeu commence.

La limite du jeu est lorsque l'utilisateur gagne la partie c'est-à-dire il trouve le mot ou lorsqu'il obtient 6 erreurs et l'ordinateur le demande si il veut rejouer ou non.

Pour l'executer: python hangman.py
