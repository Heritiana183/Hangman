# Hangman
Jeu de devine-mot

Ceci est un projet basique du jeu hangman qui consiste à deviner le mot exact.

J'ai choisi des mots au hasard puis je les ai mis dans mon code. Ensuite, j'ai utilisé la fonction random pour les choisir aléatoirement, puis le jeu commence.

La limite du jeu est lorsque l'utilisateur gagne la partie, c'est-à-dire qu'il trouve le mot ou lorsqu'il commet 6 erreurs. L'ordinateur le demande s'il veut rejouer ou non.

Pour l'executer: python hangman.py
